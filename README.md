## Подготовка базы данных

`
python manage.py makemigrations
`

`
python manage.py migrate
`


Загрузит фильмы из файла [films.json](films.json) в бд

`
python manage.py loaddata films.json
`

## Запуск проекта

`
python manage.py runserver
`
